# average-median

## Getting Started

### Install using npm:


```
$ npm i average-median --save

or

$ yarn add average-median
```

### Basic Usage

```
const averageMedian = require('average-median');

const list = [2, 89, 45, 45, 56, 34, 100];

const averageMedian = new AverageMedian();

console.log(averageMedian.calculate(list))

```
